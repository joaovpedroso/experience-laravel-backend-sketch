-------------- LARAVEL COMANDOS ---------------------------

* Composer -v 	= Verificar se esta instalado o composer.

* composer global require "laravel/installer"		    ---> INSTALAR O LARAVEL 

* composer create-project laravel/laravel laravel "5.4.*"   ---> CRIAR UM NOVO PROJETO


- php artisan serve 		     -> Starta o Projeto
- php artisan migrate:refresh --seed -> Atualiza o banco de dados, Cria tabelas


COMPOSER INSTALL 	-> INSTALA O COMPOSER NO PROJETO
COMPOSER UPDATE		-> ATUALIZA AS DEPENDENCIAS DO PROJETO




View


Rotas 			-> Pasta Routes. Indica o caminho a ser aberto.
			*> Web.php -- Arquivo de declaração das Rotas.


Controller 		-> Açao da View. Toda view tem um controle relacionado a ela. Controle das ações.


MVC			-> Padrão de Desenvolvimento.


ENV 			-> ARQUIVOS DE CONFIGURAÇÃO DO PROJETO ( BANCO DE DADOS ).

 

MIGRATIONS 		-> ARQUIVOS DE ALTERAÇÕES, MODIFICAÇÕES.
